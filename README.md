Here's how Formula 2 calculates rankings:
Phase 1: Learn Meta-Level Statistics (from entire dataset)

Calculate empirical stats for each deck:

Adjusted wins = wins + (0.5 × ties)
Win rate = adjusted wins ÷ total games


Calculate meta-level parameters:

totalGames = sum of all games across all decks
weightedMean = Σ(win_rate × games) ÷ totalGames
variance = Σ(games × (win_rate - weightedMean)²) ÷ totalGames


Estimate Beta distribution hyperparameters:

Clamp meanEst between 0.01 and 0.99
Clamp varEst between 0.0001 and (meanEst × (1 - meanEst) × 0.9)
alphaBeta = (meanEst × (1 - meanEst) ÷ varEst) - 1
priorAlpha = max(0.5, meanEst × alphaBeta)
priorBeta = max(0.5, (1 - meanEst) × alphaBeta)


Determine adaptive Z-score based on dataset size:

If totalGames > 50,000: z = -1.96 (97.5th percentile)
Else if > 10,000: z = -1.75
Else if > 5,000: z = -1.645 (95th percentile)
Else: z = -1.5 (conservative)



Phase 2: Calculate Each Deck's Ranking

For each individual deck:

adjWins = wins + (0.5 × ties)
adjLosses = losses + (0.5 × ties)
n = wins + losses + ties


Calculate posterior parameters:

postAlpha = priorAlpha + adjWins
postBeta = priorBeta + adjLosses


Calculate posterior mean (shrinkage estimator):

posteriorMean = postAlpha ÷ (postAlpha + postBeta)


Calculate lower bound of credible interval:

postVariance = (postAlpha × postBeta) ÷ ((postAlpha + postBeta)² × (postAlpha + postBeta + 1))
postStdDev = √postVariance
lowerBound = max(0, posteriorMean + (adaptiveZ × postStdDev))


Convert to final strength score (1-100 scale):

hierarchicalWinPct = lowerBound × 100
strength2 = ((hierarchicalWinPct - 30) ÷ 25) × 99 + 1
